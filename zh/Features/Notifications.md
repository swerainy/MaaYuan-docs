# 通知设置<Badge type="tip" text="简单" />

::: tip
MaaYuan 支持任务完成后发送通知，可使用钉钉或邮箱进行测试与接收。
:::

## 一、钉钉通知

1. 在电脑端钉钉建群。
2. 进入“群设置 → 机器人 → 添加机器人 → 自定义”。
3. 安全设置选择“加签”。
4. 复制 `Webhook` 地址与密钥。
5. 打开 MaaYuan → 设置 → 外部通知 → 开启钉钉通知。
6. 将以下信息分别填入：

| MaaYuan 字段 | 填写内容 |
| --- | --- |
| `机器人 Token` | `Webhook` 地址 |
| `钉钉 Secret` | 机器人密钥 |

7. 如有需要，可填写自定义通知内容。
8. 点击“发送测试”验证。

## 二、邮箱通知

1. 登录邮箱网页端。
2. 进入“账号与安全”相关设置。
3. 开启 `POP3 / IMAP / SMTP / Exchange / CardDAV` 等服务。
4. 生成授权码。
5. 把邮箱账号和授权码填入 MaaYuan 对应位置。
6. 点击“发送测试”确认可正常接收邮件。

| MaaYuan 字段 | 填写内容 |
| --- | --- |
| `机器人 Token` | 邮箱账号 |
| `邮箱 Secret` | 生成的授权码 |

## 三、使用建议

- 先发测试，确认通道可用，再正式依赖通知。
- 若只是本地盯进度，外部通知不是必需项。
- 涉及邮箱授权码时，不要随意分享配置截图。
