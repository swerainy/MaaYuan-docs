# 作业功能<Badge type="warning" text="普通" />

::: tip
本文整理 MaaYuan Share 作业站、神秘代码导入、本地导入、作业制作与运行注意事项。
:::

## 一、作业站能做什么

- 查找作业
- 复制神秘代码
- 下载 `JSON` 文件
- 批量导入作业集
- 制作、修改和分享作业

## 二、查找与下载作业

1. 可通过搜索栏按标题、描述、神秘代码筛选。
2. 也可以按关卡分类查找。
3. 下载方式有两种：
   1. 复制神秘代码
   2. 下载 `JSON` 文件

## 三、导入作业

### 本地导入

1. 下载 `JSON` 文件。
2. 在 MaaYuan 中选择“本地导入”。
3. 选中文件即可。

也可以直接把文件放到：

```txt
resource\copilot-cache
```

::: warning
不要把文件放到原始 `copilot` 文件夹，否则可能导致资源加载失败。
:::

### 神秘代码导入

1. 在作业站复制神秘代码。
2. 打开 MaaYuan 自动战斗界面。
3. 粘贴代码并选择导入方式。

- 单份作业：左键“使用神秘代码”
- 作业集：右键“使用神秘代码”

## 四、运行作业的前提

- 已导入并激活作业
- 队伍与命盘按作业要求调整好
- 在正确页面开始任务
- 右侧自动 / 手动 / 倍速面板保持展开

## 五、作业制作

### 基础信息

制作作业时通常需要填写：

- 关卡
- 标题
- 描述
- 作业来源
- 延迟设置
- 可见范围

### 动作编辑支持

- 动作链视图
- 类表格视图
- 添加等待
- 切换目标
- 吕布形态切换
- `SP` 技能
- 开启自动战斗
- 多种重开条件

## 六、录制作业

`v2.0.0-beta.3` 及以上版本支持录制作业：

1. 进入录制作业界面。
2. 点击“开始录制”。
3. 使用“操作记录仪”悬浮窗记录动作。
4. 如果模拟器连接正确，操作会同步到模拟器中。

## 七、作业相关常见问题

- 导入后显示“未激活”：需要手动点击对应作业加载。
- 导入成功但无法运行：先核对开始页面是否正确。
- 神秘代码导入失败：多半是作业站网络问题，可稍后重试或改用本地导入。
- 资源加载失败：优先确认文件目录和版本是否匹配。
