# 任务总览<Badge type="warning" text="普通" />

::: tip
本文汇总参考手册中的任务能力，帮助你先建立对 MaaYuan 的整体认识。
:::

## 一、任务运行的共同行为

- 大部分任务遇到体力不足或次数不足时，会自动停止并尝试继续后续任务。
- 开启无限循环和自动吃鸟食时，也不会消耗白金币购买鸟食。

## 二、日常一条龙包含的内容

- 模拟器、游戏自启动
- 鸢报四连（日常专用）
- 领月卡福利
- 每日进膳
- 据点日常
- 观星
- 历练
- 白鹄行动
- 地宫限定
- 心纸营建
- 其他日常收尾

## 三、其他护肝功能

- 刷 `6-24 / 7-15`
- 刷到体力耗尽或指定循环次数
- 刷【宇宙匣】三千世界成就
- 自动刷洞窟
- 自动走历险
- 自动抄作业（测试版）
- 鸢报四合一

## 四、鸢报四合一说明

1. 会轮流做已激活的鸢报事件，尽量不浪费时间。
2. 鸟食数量不足的事件会自动跳过。
3. 为了循环检查，决策事件和没有建材箱的物资支援会强制做掉。
4. 如果不想“浪费”鸡炙，请改用单独的“待办公务”任务。

## 五、预设切换

官方提供的预设包括：

- 代号鸢日常模板
- 如鸢日常模板
- 特色功能合集
- 新版全部功能

你也可以按需在任务列表里添加、删除或切换任务，为不同账号做不同预设。
