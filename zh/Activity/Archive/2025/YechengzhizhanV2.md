# 邺城之战 V2<Badge type="tip" text="2025" />

- 版本：`v0.9.13-beta.8`
- 支持自动导航，也可在进行中的攻取关卡内启动，会自动接管当前进度继续刷。
- `beta7` 适配“三棱镞已满”的弹窗。
- 支持全部关卡、全部战车选择。
- Maa 会强制开启自动战斗，请提前准备好编队；想刷藏兵洞的，也需要提前选好首发队员。
- 支持升级天赋树（不升级 / 升攻击 / 升生命），支持收集壮行酒。
- 支持自动购物，金 / 紫 / 蓝 buff 识别精度可调。
- 支持路径优先级选择（刷煤 / 收集 buff）。
- 走不动路时，可在“任务设置 - 高级”中调整识别精度（`0~1` 之间的实数）。
- 如识别不到应去的高亮节点，可把对应节点的识别精度调低；如一直在点灰色节点，可把对应节点的识别精度调高。
- 循环刷图建议配置（参考云梦）：设置好后请关闭代号鸢再启动任务。
- 注：新版 MuMu 需要先创建单个模拟器的桌面快捷方式。
