# 江东万里船 V3<Badge type="tip" text="2025" />

- 启动后会自动导航进入活动界面，也可在未完成水战关卡内启动，会自动接管当前进度继续刷。
- Maa 会强制开启自动战斗，请提前准备好编队。
- 常规设置包括：关卡选择、游船选择、路径优先级（刷材料 / 集 buff / 通关）、船歌优先级、是否进行漂流罐及迷雾战斗、是否自动购物、延时设置、超时后行为（关闭游戏 / 重开关卡）、战斗失败后行为（停止任务 / 重开关卡）、战斗超时时间。
- 高级设置包括：各级精度自定义。识别不到某个节点，可把对应识别精度调低；若一直点灰色节点，可把对应灰色内容的识别精度调高。
