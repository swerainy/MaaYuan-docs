# 朝歌之战<Badge type="tip" text="2025" />

- 本功能只是囤资源小助手，暂不支持推主线。
- 支持囤粮 / 爬塔。
- 支持自动处理叹号 / 自动迎战。
- 支持选择刷突破材料（低级 / 高级 / 都不刷）。
- 需要挂几回合，就把重复次数设为几。
- 运行后会按照“叹号事件 → 资源战斗 → 突破素材战斗”的顺序消耗行动力，并自动推进回台。
- 建议把界面右上、右下的面板都收起来。
- 已配备自动导航，将强制开启游戏内置自动战斗。
- 如果想节约时间，可以不设置选项里的“自动补兵”和“指定队伍”。
- 附：会开自动过所有主线战斗的 `56` 回合流程攻略与进度追踪网页。
